# Project Information

## Technology Required

- Node.js
- tRPC
- Typescript
- TypeORM

## Technology Used

- [T3 Stack](https://create.t3.gg/)
  - Next.js (**Node.js**)
  - **tRPC**
  - **React**
  - **Typescript**
  - Tailwind.css
- PostgreSQL
- **TypeORM**

## Functionality

- User management application
  - List users
    - First name (required)
    - Last name (required)
    - Email address (required)
    - Favourite food (not required)
  - Add users
  - Delete users

## Process

1. Install dependencies
   - Install PostgreSQL ✅
   - Create new T3 app ✅
   - Install TypeORM ✅
2. Configure database
   - Create database
   - Create user table
3. Create backend
   - `GET`: Get all users
   - `POST`: Create a new user
   - `DELETE`: Delete a user
4. Create frontend
   - List all users
   - Add a new user
   - Delete a user
